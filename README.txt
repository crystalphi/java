Demo projects based on Hadoop 2.4.1 and Mahout 0.9.
----------------------------------------

The project "myHadoop" is generated by cmd:
mvn archetype:generate -DarchetypeGroupId=org.apache.maven.archetypes -DgroupId=org.vedasky.myhadoop.mr -DartifactId=myHadoop -DpackageName=org.vedasky.myhadoop.mr -Dversion=1.0-SNAPSHOT -DinteractiveMode=false

The project "myMahout" is generated by cmd:
mvn archetype:generate -DarchetypeGroupId=org.apache.maven.archetypes -DgroupId=org.vedasky.mymahout -DartifactId=myMahout -DpackageName=org.vedasky.mymahout -Dversion=1.0 -DinteractiveMode=false
